Function cqnToOData

  • cqnToOData(select: CqnSelect, opts?: ODataOptions): string

    Converts a CAP CQN SELECT object into an OData query string.

    Supported parameters:

    • $select (from columns)
    • $filter (from where)
    • $orderby (from orderBy)
    • $top / $skip (from limit)
    • SuccessFactors dates: fromDate, toDate or asOfDate (exclusive)

    Rules:

    • If asOfDate is provided, it is exclusive and fromDate/toDate are ignored.
    • Otherwise, fromDate and/or toDate can be provided independently.

    Parameters

    • select: CqnSelect

      CAP CQN SELECT subset

    • Optionalopts: ODataOptions

      Optional OData and SuccessFactors options

    Returns string

    Query string with parameters joined by '&'
